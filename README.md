# WallpaperApp

Приложение для установки обоев на экран телефона.
Использовано [PixabayAPI](https://pixabay.com/api/docs/ ) для выбора картинки.


### Стек:
- Kotlin
- Single Activity
- XML
- Retrofit
- Room
- Coroutines Flow
- Navigation Architecture Component
- Dagger
- Paging3
